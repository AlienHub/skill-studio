# Changelog

## 0.6.0 - 2026-05-09

### 新功能
- 新增 Agent Context Catalog 统计：按 Agent 估算常驻 skill 数量和 catalog token 体量。
- Skill 详情页新增元数据级别的 catalog token 展示，便于快速判断单个 skill 的常驻成本。

### 改进
- 首页常驻上下文收敛为更轻的 Agent 摘要行，突出常驻 skill 数和 token 量级。
- 将 `~/.agents/skills` 视为共享 skill 源合并进主流 Agent catalog，并避免重复计数。
- catalog 统计现在会优先保留 Agent 本地目录中的 skill 副本，再合并共享目录来源。

## 0.5.1 - 2026-05-08

### 新功能
- 新增 skill 分享能力，可将当前真实来源以软链接形式分享到其他已配置 Agent。
- 新增导出 ZIP，可直接打包当前真实来源用于手动分享。
- 新增同内容来源治理，可将内容完全相同但位置不同的真实来源切换为软链模式。

### 改进
- 软链治理操作保留本地优先和可恢复策略：真实目录会先进入系统废纸篓，再创建新的软链接。

## 0.5.0 - 2026-05-08

### 改进
- 产品名更新为 Skill Grove，作为 v0.5 品牌与精致度阶段的第一版命名。
- 同步更新 app 标题、Tauri 产品名、README 首屏文案和发布说明。
- 同步更新内部包名、crate 名、bundle identifier、GitHub Release 地址和本地存储 key。
- 将版本号提升到 0.5.0，准备后续品牌、图标和 macOS 体验打磨。

## 0.4.0 - 2026-05-08

### 新功能
- 详情页增加 sticky mini header，长说明滚动时仍能看到当前 skill、来源位置和收藏入口。
- 新增 Pinned Skills / Recently Viewed 本地整理状态，并接入 Library 排序与列表标记。
- 设置页新增默认打开方式，可选择文件管理器、编辑器或 IDE 作为来源按钮的默认动作。

### 改进
- Markdown preview 优化标题、列表、代码块、表格和 frontmatter 展示。
- Home 页面收敛为更简洁的起始卡片，弱化数据看板感，只保留入口、状态和最近变化。
- 统一加载状态为 Ripple 动画，用于更新检查、安装、目录扫描和来源移除等操作。
- 阅读与整理能力继续保持本地优先，只写入浏览器 / WebView 的本地偏好。

## 0.3.0 - 2026-05-08

### 新功能
- 新增 Since Last Visit 本地快照，用当前扫描结果和上次快照对比最近变化。
- 新增 Home 页面，集中展示欢迎状态、上次打开后的变化和轻量整理建议。
- 新增 Library 筛选：全部、多来源、有变体、最近变更、值得看看，并默认收进搜索框旁的筛选菜单。
- 新增 Worth a Look 建议，先覆盖多内容版本、内容不一致、描述缺失、说明过长和自定义来源。
- Skill 详情页新增最近变化与值得看看提示。

### 改进
- 将最近变化和建议保持为本地浏览器 / WebView 数据，不读取会话记录，不上传数据。
- 首页与筛选数字统一按 skill 数量展示，避免建议条数和筛选结果数量不一致。

## 0.2.2 - 2026-05-08

### 改进
- 开始规划 v0.3「回访理由与轻量整理」版本设计，明确 Since Last Visit、轻量状态条、筛选和建议机制的产品边界。
- 更新 roadmap，让 v0.3 从功能清单推进到可执行的设计切片。

## 0.2.1 - 2026-05-07

### 新功能
- 新增外观偏好，支持跟随系统、浅色和深色模式。
- 新增界面语言偏好，支持默认语言、中文和 English。

### 改进
- 深色模式统一调整为中性黑灰色，去除紫蓝底色，并提升暗色边框可见度。
- 多处核心界面文案接入中英文翻译，为后续完整国际化打基础。

### 修复
- 修复 Tauri app 透明标题栏在浅色和深色模式切换后背景不同步的问题。
- 修复深色模式下 app 顶部原生导航区域颜色异常的问题。

## 0.2.0 - 2026-05-07

### 新功能
- 重构技能详情的来源视图，支持按内容版本聚合展示多个 Agent 来源。
- 新增当前来源切换菜单，支持按 Agent 或路径搜索来源。
- 新增当前来源操作区，可直接用访达打开目录，或通过下拉菜单选择 IDE/编辑器打开。
- 支持清理来源：软链可直接删除，真实来源目录会放入 macOS 废纸篓。

### 改进
- 多内容版本卡片改为一行两个展示，单版本或单来源时自动占满宽度。
- 优化来源信息布局，将当前来源路径和操作按钮合并到同一信息区。
- 精简来源状态文案，降低操作按钮的视觉权重。

### 修复
- 修复浏览器开发环境下来源清理接口缺失导致的失败。
- 修复来源切换和来源操作下拉层级被信息表格遮挡的问题。
