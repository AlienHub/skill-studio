# Skill Grove v0.6.0

Released: 2026-05-09

## Highlights

- Added Agent Context Catalog estimates so each Agent shows how many skills are expected to stay resident and roughly how many catalog tokens they occupy.
- Added per-skill catalog token display in metadata, making resident context cost visible without opening a separate analysis panel.
- Merged shared `~/.agents/skills` entries into mainstream Agent catalog estimates and deduplicated overlapping local copies.
- Simplified the Home page catalog summary into a compact Agent-by-Agent overview.

## macOS

- This release publishes a macOS DMG installer only.
- Download the `.dmg` asset below and drag Skill Grove into Applications.
- Users on v0.5.1 can use the in-app updater to install this release.
