# Skill Studio v0.1.3

Released: 2026-05-07

## Highlights

- Fixed CLI Agent installation detection when Skill Studio is launched as a packaged macOS app.
- Codex, OpenCode, OpenClaw, Hermes, and other CLI-only Agents now resolve commands from common user install paths and the login shell PATH.
- Codex can also be detected from the macOS app bundle when available.

## macOS

- This release publishes a macOS DMG installer only.
- Download the `.dmg` asset below and drag Skill Studio into Applications.
- Users on v0.1.2 can use the in-app updater to install this release.
